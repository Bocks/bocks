# bocks

Code Annotation

## Getting Started

### Development

* Fork and clone this repo
* Start up mongo
* `npm install`
* `npm start`
* Browse to [http://127.0.0.1:1337](http://127.0.0.1:1337)

See [deployment.md](docs/deployment.md) for how to deploy.
